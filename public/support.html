tidystats Word Add-in

Support & Frequently Asked Questions

What is tidystats?

tidystats is an R package that lets you save the output of statistical tests into a single file. The tidystats Word add-in lets you insert and update those statistics directly in your Word document.

What do I need to use the add-in?

You need an internet connection and a compatible version of Microsoft Word:

  • Word on the web
  • Word on iPad
  • Word on Mac (Microsoft 365, 2016, or 2019 and later)
  • Word on Windows (Microsoft 365, 2016, or 2019 and later)

How do I install the add-in?

The add-in is available in the Office Add-in Store. Open a Word document, go to the Insert tab, click Add-ins, and search for "tidystats".

How do I create a tidystats file?

You need the tidystats R package. After running your statistical analyses in R, use write_stats() to save the results to a .json file. See the introduction vignette for a step-by-step guide.

If you want to try the add-in without running R first, you can download this example file.

How do I insert statistics into my document?

  1. Open the add-in from the Insert tab.
  2. Click Upload statistics and select your tidystats .json file.
  3. Browse or search the list of analyses.
  4. Click an analysis to expand it, then click the + icon to insert a statistic.

How do I update statistics after re-running my analyses?

Upload a new tidystats file (with the same analysis identifiers as before), then click Update statistics in the Actions section. All previously inserted statistics will be updated automatically.

Which R packages and statistical tests are supported?

tidystats has built-in support for many R packages including stats, lme4, BayesFactor, brms, afex, and more. See the supported functions vignette for a full list. You can also report statistics from unsupported functions using custom_stats().
